HeiaHeia Feed Widget is a widget that brings a piece of HeiaHeia to your own blog.

== Description ==

HeiaHeia Feed Widget allows HeiaHeia users to embed a user, a group or an organisation feed on the sidebar of the use.

**Features**

* Users can embed one of the following feed types: User, Group or Organisation.
* Widget options for width and height.
* Widget options for language, 11 languages are availbe – Dansk, Slovenščina, Español, Français, Italiano, Svenska, Suomi, Norsk, Deutsch, English, Русский.
* Widget options for measurement units – Metric or Imperial.
* Widget options for content type – "Latest Activities" or "Top 5 Sports".

== Installation ==

1. Unzip and upload the HeiaHeia Widget directory to `/wp-content/plugins/`.
2. Activate the plugin through the **Plugins** menu in WordPress.
4. Visit [HeiaHeia User Settings](https://www.heiaheia.com/settings/edit/) page to check that plugins are enabled.
5. Visit [HeiaHeia User IDs](https://www.heiaheia.com/settings/user_ids/) page to get an ID you are interested for.
3. Visit **Appearance > Widgets** to setting up the widget.

== Frequently Asked Questions ==

= Where can I get an ID to fill "Object ID" field on the widget settings page? =

"Object ID" is an ID of User, Group or Organisation registered on HeiaHeia. Visit [HeiaHeia User IDs](https://www.heiaheia.com/settings/user_ids/) page to get an ID you are interested for.
